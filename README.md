# ghostwriter

SillyTavern 입력창의 유저 초안을 버튼 한 번으로 "유저 시점의 3인칭" 문장으로 대필하는 테스트용 확장입니다.

## 현재 기능

- 입력창(`#send_textarea`)의 현재 내용을 읽습니다.
- SillyTavern에서 현재 활성화된 API 연결을 그대로 사용합니다.
- 원문 의도, 행동, 감정은 유지하고 유저가 조종하는 인물의 3인칭 문장으로 다시 씁니다.
- `{{user}}`, `{{User}}`, `<USER>`를 대필 대상자로, `{{char}}`, `<BOT>`을 금지 대상자로 명시합니다.
- `{{user}}`와 `{{char}}`의 프로필 정보를 참고해 성별에 맞는 지시대명사와 호칭을 선택하도록 지시합니다.
- 원문에 없는 캐릭터 이름을 새로 만들거나 현재 채팅 캐릭터 이름을 주어로 가져오지 않도록 지시합니다.
- 결과는 입력창의 원문을 바로 덮어씁니다.
- 채팅별 최신 대필 기록을 입력창 바로 위 패널에 보여줍니다.
- 히스토리 패널에는 시간과 대필 결과만 한 줄로 표시하고, 원문은 표시하지 않습니다.
- 펼친 히스토리 항목의 키보드 아이콘을 누르면 해당 결과를 입력창에 다시 적용합니다.
- `전체보기`를 누르면 긴 대필 결과를 펼쳐 볼 수 있고, 다시 접을 수 있습니다.
- 히스토리 패널 상단의 재대필 버튼으로 직전 입력 원문을 현재 설정으로 다시 대필할 수 있습니다.
- 영어로 출력된 히스토리 항목은 펼친 상태에서 한국어 번역을 패널 안에서만 확인할 수 있습니다.
- 패널 닫기 버튼을 누르면 현재 채팅의 히스토리 패널을 숨깁니다.
- 히스토리는 브라우저 localStorage에 채팅별로 최신 3개만 저장합니다.
- 확장 설정에서 SillyTavern API 연결에 저장된 연결 프로필 중 대필용 프로필을 선택할 수 있습니다.
- 대필 실행 전 지정 프로필로 전환하고, 대필이 끝나면 원래 프로필로 복귀합니다.
- 연결 프로필 확인 또는 전환에 실패하면 별도 오류 문구를 보여줍니다.
- 대필 톤, 출력 언어, 길이, 참고할 최신 메시지 수를 설정에서 선택할 수 있습니다.
- 말투와 분위기를 슬라이더로 조절할 수 있고, 값은 채팅별 현재 유저 페르소나마다 따로 저장됩니다.
- 시제는 설정 없이 과거형으로 고정합니다.
- 최신 메시지는 장면, 관계, 분위기 참고용으로만 사용하고 대필 대상은 입력창 원문으로 제한합니다.
- 대필 옵션 기본값 초기화 버튼으로 톤, 출력 언어, 길이, 참고 메시지 수를 기본값으로 되돌릴 수 있습니다.
- 대필 중에는 귀신 버튼 아이콘이 커졌다 작아지는 애니메이션으로 상태를 표시합니다.

## 파일 구조

```text
ghostwriter/
  manifest.json  # SillyTavern이 확장을 인식하는 설정 파일
  index.js       # 버튼 추가, 입력 읽기, 대필 요청, 결과 반영 담당
  style.css      # 버튼, 설정 패널, 입력창 위 히스토리 패널 스타일
  README.md      # 사용법과 구조 설명
```

## 설치 위치

이 폴더를 SillyTavern의 서드파티 확장 폴더에 넣으면 됩니다.

```text
SillyTavern/public/scripts/extensions/third-party/ghostwriter
```

## 주로 수정할 곳

`index.js`의 `DEFAULT_SYSTEM_PROMPT`를 바꾸면 대필 규칙을 바꿀 수 있습니다.

`index.js`의 `setButtonIcon()`을 바꾸면 버튼 아이콘을 바꿀 수 있습니다.

`style.css`의 `.ghostwriter-button`을 바꾸면 버튼 모양을 조정할 수 있습니다.

확장 설정의 `대필용 연결 프로필(API)` 드롭다운에서 SillyTavern Connection Profiles에 저장된 프로필을 선택하면 됩니다. `현재 연결 그대로 사용`을 선택하면 프로필 전환 없이 현재 API 연결로 대필합니다.

설정에서 고를 수 있는 대필 옵션:

```text
대필 톤: 기본 / 담백함 / 섬세함 / 문학적 / 짧고 간결함
출력 언어: 한국어 / 영어
길이:
  짧게: 1~2문장, 원문과 비슷한 분량
  보통: 2~4문장, 행동/감각/감정 묘사를 조금 보강
  길게: 4~7문장, 현재 상황에 어울리는 짧은 대사 추가 가능
참고할 최신 메시지:
  참고 안 함: 0개
  적게: 2개
  기본: 4개
  많게: 6개
  최대: 10개
페르소나 문체 조절:
  말투: 격식 - 약간 격식 - 기준 - 약간 구어체 - 구어체
  분위기: 진지함 - 약간 진지함 - 기준 - 약간 가벼움 - 장난기
```
